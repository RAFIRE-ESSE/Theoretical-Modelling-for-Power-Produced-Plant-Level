# Theoretical-Modelling-for-Power-Produced-Plant-Level
Built a solar power prediction pipeline using stacked ML ensembles (LightGBM, XGBoost, CatBoost) and a Bidirectional LSTM, achieving R² of 0.9969. Modelled five physics-based power loss types — angle, soiling, temperature, cloud cover, and inverter efficiency — for granular plant-level loss attribution.
